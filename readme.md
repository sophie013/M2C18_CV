# Portafolio Profesional - Sophia López Riquelme

Este proyecto es un Currículum Vitae interactivo desarrollado para la asignatura de Desarrollo Web. El sitio destaca mi trayectoria profesional en Gastronomía Internacional y Administración de Empresas, marcando mi transición hacia el desarrollo Front-End.

## Tecnologías Aplicadas
- **HTML5**: Estructura semántica avanzada utilizando etiquetas como `<header>`, `<main>`, `<section>` y `<footer>` 
- **Bootstrap 5**: Implementación de un diseño responsivo (Mobile First) mediante el sistema de rejilla (Grid System) y componentes modernos.
- **jQuery**: Gestión de eventos del DOM, animaciones personalizadas e interactividad fluida.
- **CSS3**: Estilos personalizados, manejo de variables para temas y diseño de cursores interactivos.

## Funcionalidades del Sitio
1. **Interruptor de Tema**: Sistema dinámico que permite alternar entre modo claro y modo oscuro 
2. **Validación en Tiempo Real**: El formulario de contacto incluye una lógica  que verifica el formato del nombre y correo (aceptando solo  extensiones `.cl` y `.com`) antes de permitir el envío.
3. **Soft Scroll & Navegación Activa**: Desplazamiento suave entre secciones y detección automática de la ubicación del usuario en el menú lateral.
4. **Interactividad Visual**: Uso de la librería `Typed.js` para efectos de escritura y un cursor personalizado que sigue el movimiento del mouse.

## Estructura del Proyecto
- `index.html`: Archivo principal con la estructura semántica.
- `css/style.css`: Estilos personalizados y ajustes de diseño.
- `js/script.js`: Lógica de validación, cambio de tema y efectos.
- `assets/`: Carpeta que contiene imágenes, documentos descargables e iconos.
